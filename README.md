# TinyApp
The app will allow users to shorten long URLs much like TinyURL.com and bit.ly do.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting started

- Install all dependencies
- Run the development web server using the 'node express_server.js' command.
